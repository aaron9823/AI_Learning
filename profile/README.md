# AI_Learning Project

This project contains a simple business card for 홍길동, a QA Engineer. The business card is designed to be visually appealing and includes essential contact information along with a placeholder for a QR code.

## Project Structure

- **profile.html**: Contains the HTML structure for the business card, including the name, title, phone number, and a placeholder for a fake QR code.
- **profile.css**: Contains the CSS styles for the business card, ensuring a well-organized layout with a border around the card.
- **profile.js**: Intended for future JavaScript functionality, such as adding interactivity or generating a QR code.

## How to View the Business Card

1. Open the `profile.html` file in a web browser.
2. The business card will be displayed with the provided information.

## Additional Notes

- You can modify the CSS in `profile.css` to change the appearance of the business card.
- Future enhancements can be added in `profile.js` to improve functionality or interactivity.
